# GHX Website - GRYTTE Financial Services

A high-converting website for the GHX 12-Month Transformation Program, a financial coaching service for young men in Cleveland, Ohio.

## Project Status

**Phase:** Planning/PRD
**Target Launch:** Q1 2025
**Location:** Cleveland, Ohio

## What is GHX?

GHX is a 12-month program combining credit repair, financial coaching, and identity transformation. The program uses a "belt system" (Red → Yellow → Green → Black) to track client progress toward specific goals:

- 720+ credit score
- $10,000 saved
- 20% income increase
- Personal brand launched

## Recommended Tech Stack

| Option | Best For | Notes |
|--------|----------|-------|
| Webflow | Speed to market | No-code, hosting included, ~$18/month |
| WordPress + Elementor | Budget-conscious | More flexible, requires maintenance |
| Next.js + TailwindCSS | Custom features | Best for client portal, requires developer |

## Site Structure

```
/              Homepage
/about         Our Story, Why Cleveland, Team
/program       Belt System, Timeline, What's Included
/results       Success Stories, Case Studies
/pricing       Standard/Enhanced/Premium tiers
/resources     Blog, Lead Magnets
/faq           Categorized FAQ
/apply         Pre-qualification + Calendly
/login         Client Portal
```

## Required Integrations

- Calendly (discovery calls)
- ConvertKit or Mailchimp (email marketing)
- HubSpot or Pipedrive (CRM)
- Google Analytics 4 + Tag Manager
- Hotjar (heatmaps)

## Success Metrics

- **Conversion Rate:** 15-25% visitors → discovery calls
- **Monthly Visitors:** 500-1,000 in first 90 days
- **Bounce Rate:** <50%
- **Session Duration:** 3-5 minutes
